Research

concordant Crop Sequence Boundaries

The concordant Crop Sequence Boundaries is a field boundary dataset which is an improved version of the Crop Sequence boundaries which relies on the similarity of the crop sequences to merge field fragments together, producing more stable field boundaries through time.

If you want to try using the dataset, you can download it here. I would not suggest its use for areas which have poor crop detection in the Cropland Data Layer, eveywhere else should be fairly ok. I would suggest to use the field boundaries from Yan and Roy 2016 to confirm your results.
